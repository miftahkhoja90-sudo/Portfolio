 # School Management System - Multi-Page Website

This is a fully responsive **School Management System** front-end project built using **HTML, CSS, JavaScript, and jQuery**.

## Features

- Modern, clean UI with responsive layout (mobile, tablet, desktop)
- Multi-page structure with shared header, navigation, and footer
- Student and Teacher dashboards
- Admission, Login, Register, and Contact forms with client-side validation
- Responsive tables for attendance, results, and class schedules
- Gallery page with lightbox effect
- jQuery-powered interactions: mobile menu, smooth scrolling, back-to-top button, carousel, modal, and more

## Structure

- `index.html` – Home
- `about.html` – About School
- `courses.html` – Courses / Classes
- `teachers.html` – Teachers / Staff
- `dashboard-student.html` – Student Dashboard
- `dashboard-teacher.html` – Teacher Dashboard
- `admission.html` – Admission Form
- `attendance.html` – Attendance Page
- `results.html` – Results Page
- `gallery.html` – Gallery
- `events.html` – Events / News
- `contact.html` – Contact Us
- `login.html` – Login Page
- `register.html` – Register Page
- `css/style.css` – Base styles
- `css/responsive.css` – Responsive & media queries
- `js/script.js` – JavaScript & jQuery logic
- `jquery/jquery.min.js` – jQuery library (downloaded separately)
- `images/` – Images and icons

## Getting Started

1. Download the repository into a folder named `school-management-system` (or use `E:\project2` in Cursor).
2. Download the latest **jQuery** (e.g. 3.x minified) and save it as:

   - `jquery/jquery.min.js`

3. Open `index.html` in your browser.

No backend is required; this is a purely front-end demo project.
